# LeetCode Solutions

This folder contains my solutions to LeetCode problems in C.

## Solution Header Template

```c
/*
Problem: <Problem Name>
LeetCode #<Number>

Approach:
<Brief explanation of the solution>

Time Complexity: O(...)
Space Complexity: O(...)
*/
```

## Naming Convention

```
<Category>/<ProblemNumber>_<ProblemName>.c
```

Examples:

```
Arrays/0001_TwoSum.c
Math/0009_PalindromeNumber.c
Strings/0013_RomanToInteger.c
Stack/0020_ValidParentheses.c
```

## Notes

- Focus on understanding the approach rather than memorizing code.
- Include time and space complexity for every solution.
- Prefer optimized solutions whenever possible.
